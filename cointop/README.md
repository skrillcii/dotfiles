# CoinTop

# Prerequisite
* Ubuntu 20.04
* Golang 13.8 linux/amd64

## Installation
Make sure to have go (1.12+) installed, then do:
```
go get github.com/miguelmota/cointop
```

Configuration file
```
$ ln -s ~/dotfiles/cointop/config.toml ~/.config/cointop/config.toml
```
